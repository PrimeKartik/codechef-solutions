## 📚 Key Learnings

**1. Another Card Game Problem**
* Ceiling division for `x / y` can be cleanly and efficiently implemented in C using integer arithmetic as `(x + y - 1) / y`, which is perfect for O(1) greedy grouping problems.

**2. Sum of palindromic numbers**
* Reversing an integer mathematically (`rev = rev * 10 + temp % 10`) is a robust, allocation-free method to check numeric palindromes without the overhead of string conversion.

**3. Maximise XOR**
* For bitwise XOR maximization with free sequence rearrangements, counting total set bits across both arrays allows you to mathematically deduce the exact number of `1`s in the result without simulating the actual rearrangements.
