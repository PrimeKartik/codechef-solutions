## 📚 Key Learnings

**1. Palindromic substrings**
* Complex string palindrome problems can sometimes be mathematically reduced; realizing that any single common character between two strings forms a length-2 palindrome completely bypasses the need for generating actual substrings.

**2. Disabled King**
* For restricted grid traversal where horizontal movement is banned but diagonals are allowed, the optimal path is purely dictated by the parity (even/odd nature) of the grid dimensions, allowing for an $O(1)$ mathematical solution.

**3. Ada and Dishes**
* To minimize the maximum sum across two parallel processors (burners), sorting the elements in descending order and greedily assigning the next largest item to the processor with the currently lowest load is an incredibly effective scheduling strategy.
