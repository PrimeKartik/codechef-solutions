## 📚 Key Learnings

**1. Chef and His Apartment Dues**
* Using a state flag (`first`) that triggers mid-loop is an efficient `O(N)` way to apply a permanent condition (like late fines) to all subsequent elements without needing a second pass over the array.

**2. Swapping Chefs Way**
* When limited to swapping symmetric opposite indices `(i, N-i-1)`, greedily placing the smaller character at the front produces the absolute minimum possible lexicographical string, reducing the entire problem to a simple post-swap `is_sorted` check.

**3. Subanagrams**
* The longest common subanagram across multiple strings is simply the intersection of their character frequencies. Tracking the global minimum count for each letter across all strings mathematically guarantees both the longest and valid sequence.
